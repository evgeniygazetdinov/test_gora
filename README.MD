IMAGE API ON GO

***************************************

        INSTALL AND RUN:

***************************************

IN FIRST RUN BEFORE USE:

  ./install.sh

=======================================

FOR RUN:

  go run api.go


***************************************

            USAGE:

***************************************

CHECK ALL

***************************************

http://localhost:8080/

***************************************

TO ADD USE QUERY LIKE THIS

***************************************

http://localhost:8080/add_image/?image_path={YOUR_PATH}

***************************************

FOR DELETE

***************************************

http://localhost:8080/delete_image/?id={YOUR_ID}
